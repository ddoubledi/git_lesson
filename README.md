# git_lesson
Git lesson overview


# create repository
```sh
git init
```

# Статус
```sh
git status
```

# Додавання файлу(файлів)
```sh
git add {file_name}
```

# Коміт
```sh
git commit --m "Commit message"
```

# Коміт із додаванням всіх файлів
```sh
git add .
git commit -m "Commit message"
```

```sh
git commit -am "Commit message"
```

# git log
```shell
git log
```
